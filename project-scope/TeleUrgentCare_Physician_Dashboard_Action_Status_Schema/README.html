Safety Design Principles
19
1. Hard-stop red flags should prevent completion and route to emergency/in-person care.
20
2. Clarification should be structured and time-bound.
21
3. Prescription sending must remain physician-controlled.
22
4. Patient-facing messages must stay plain-language and non-diagnostic unless clinician intentionally edits.
